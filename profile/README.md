## Overview

BetterDisplay Advanced is an advanced display management application that provides custom resolution creation, smooth brightness control for external monitors, HDR settings management, dummy display creation, and display arrangement tools that go far beyond what macOS System Settings offers. It addresses the many display-related limitations and frustrations that macOS users encounter — external monitors that only offer limited resolution options, no brightness slider for non-Apple displays, HDR that cannot be configured per-application, and the inability to create virtual displays for headless servers.

Custom resolution management creates and applies display resolutions that macOS does not natively offer, including HiDPI resolutions for external monitors that appear blurry at their default scaling. The resolution management handles the common issue of non-Apple displays that lack proper HiDPI support, providing the sharp text and UI rendering that Retina displays deliver. Resolution profiles can be saved and switched quickly.

Brightness and contrast control provides smooth slider adjustment for external monitors using DDC/CI protocol, bringing the same brightness control to external displays that the keyboard brightness keys provide for built-in screens. The brightness control works with most external monitors and integrates with the macOS brightness keyboard shortcuts.

Dummy display creation generates virtual displays that appear to macOS as connected monitors, enabling headless server configurations, screen sharing with specific resolutions, and testing layouts for different display sizes. Dummy displays behave identically to physical displays from the perspective of the operating system and applications.

For users with external monitors who want the display management capability that macOS provides for built-in displays — smooth brightness control, proper HiDPI scaling, and flexible resolution options — BetterDisplay Advanced is the display tool that makes every monitor work as well as an Apple display.

## Performance

BetterDisplay balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
